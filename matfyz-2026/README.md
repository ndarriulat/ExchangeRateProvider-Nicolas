## Kdo jsme

Mews je moderní hotelový software – cloudová platforma, která řeší vše od rezervací přes platby až po check-in bez recepce. Dnes nás používá přes **15 000 hotelů, hostelů a ubytovacích zařízení ve více než 85 zemích světa**.

V roce 2023 jsme se stali teprve třetím českým **jednorožcem** – startupem s valuací přesahující miliardu dolarů. Investoři jako Goldman Sachs Alternatives, Kinnevik, EQT a Tiger Global do nás vložili přes 410 milionů dolarů.

A kde to celé začalo? Na Matfyzu. 🎓

## Naše kořeny na Matfyzu

Příběh Mews je příběhem matfyzáků. Firma vznikla, když zakladatel **Richard Valtr** otevíral rodinný hotel Emblem v centru Prahy a chtěl moderní hotel bez recepčních. Taková služba tehdy neexistovala – tak se spojil se studenty MFF UK, kteří se poznali při univerzitním projektu, a společně vytvořili řešení, které dnes využívají desetitisíce hotelů po celém světě.

Dnes Mews zaměstnává přes **40 absolventů a studentů Matfyzu** a jsme strategickým partnerem fakulty od roku 2018.

## Jak u nás vypadá engineering
 
**Stack:**
- **Backend:** C#/.NET na Azure – velký monolith jako jádro PMS + rostoucí ekosystém Atlas mikroservisů, do kterých aktivně migrujeme
- **Frontend:** React/TypeScript
- **Mobilní appky:** Flutter (Kiosk, Digital Key), React Native
- **Databáze:** Azure SQL, Redis, Azure Databricks
- **Infrastruktura:** Azure (primárně) + Terraform, GitHub Actions, Octopus Deploy
- **Observabilita:** Coralogix, Sentry, Azure Monitor
**AI-native engineering:**
 
Každý inženýr v Mews má k dispozici **Claude Code** jako primární AI coding assistant – hrazený firmou. Máme interní marketplace pluginů s Mews-specific skills pro Claude, včetně Atlas skilu, který umí scaffoldovat celé mikroservisy s vědomostí našich building blocks a guardrails. Agentic workflows jsou součástí každodenní práce, ne nice-to-have.
 
**DevOps kultura:**
- Více deployů do produkce denně
- Cross-funkční týmy vlastní svůj kód, incidenty i deployment
- SRE praktiky, [incident.io](https://incident.io), měsíční drily
